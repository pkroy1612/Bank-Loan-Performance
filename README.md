# Dashboard Screenshots

## Instructions

Add your Power BI dashboard screenshots to this folder.

### Recommended Screenshots

1. **dashboard_screenshot.png** - Full summary dashboard view
2. **kpi_overview.png** - KPI cards section
3. **trends_analysis.png** - Monthly trends charts
4. **geographic_map.png** - State-wise distribution map
5. **risk_analysis.png** - Risk assessment dashboard
6. **details_table.png** - Loan details table view

### Screenshot Guidelines

- **Resolution:** 1920x1080 or higher
- **Format:** PNG (for quality) or JPG (for size)
- **Content:** Ensure no real customer data is visible
- **Naming:** Use descriptive, lowercase names with underscores

### How to Take Screenshots in Power BI

1. Open your dashboard in Power BI Desktop
2. Navigate to the page you want to capture
3. Use Windows Snipping Tool or Snip & Sketch (Win + Shift + S)
4. Save to this folder with appropriate name
